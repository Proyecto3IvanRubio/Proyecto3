Para poder visualizar nuestro sitio web debes de seguir unos pasos.

1- Crear una base de datos con el nombre "bd_proyecto1".

2- Importar el archivo "projecte_1.sql" a la base de datos creada para crear la tabla y sus parámetros.

3- Importar el archivo "insert.sql" a la base de datos creada para insertar los datos.

4- En el archivo HTML "consulta.html" estará la consulta que realizaremos a la base de datos, tiene que estar todos los datos para que funcione.

5- En el archivo HTML "addformulario.html" estará el formulario para crear un nuevo anuncio. 

   Tiene que esta todos los datos excepto: Antigüedad , Numero de Serie y Compensación.

6- No se pueden abrir los archivos "consulta.php" y "addanucio.php" de forma directa, primero habrá que realizar los formularios si no, no funcionara.
